# DOKKAI
DOKKAIの画像配置。

# アプリ概要
ユーザーは本について語り合う部屋（bookroom）を作成し、本好きなユーザー同士で感想や解釈を深く話し合う「読書会(DOKKAI)」のように使うことができます。
本好きな人と語り合いたい人、他の人の感想を知りたい人、他の人がおすすめする本を知りたい人など、本に関することについて気軽に利用することができるアプリです。
自分の好きを言語化・確認するための機能（private,history）があり、本の世界にどっぷり浸かることができます。

# このアプリを使ってほしい人
毎日、職場と家を往復して、精力的に働いているAさん。休日は本屋に行って、新刊が出ていないかチェックして、お気に入りの本を片手にカフェでゆっくり過ごします。
Aさんは、本を読んだ後の余韻が消える前に、この本のどこが好きだったか、どんな言葉が自分の中に残ったのか、誰かとこの本について話したいな、と思っていました。
そんなAさんにDOKKAIを使ってもらい、さらに充実した日々を過ごしてほしいです、
